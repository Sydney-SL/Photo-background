## 介绍

本项目用到了：

1. Python
2. 调试
3. 脑子

本项目完全开源，且没有任何代码加密操作，如有疑虑请自行审查代码或停止使用相关文件。

## 使用方法
请在https://www.remove.bg/api 获取您的API-Key，填入软件的api输入框。
选择需要修改背景的图片，选择下方的替换颜色。
等待转换完成后，保存图片。

## 注意事项
使用代理服务器会报错。
输入图片后缀需为.png。
